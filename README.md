# 🚀 Сервис управления рассылками

Django-приложение для управления email-рассылками с системой ролей, статистикой и автоматической отправкой.

## 🌟 Возможности

### 📧 Управление рассылками
- Создание, редактирование, удаление рассылок
- Автоматическое управление статусами (Создана/Запущена/Завершена)
- Отправка рассылок по расписанию
- Ручная отправка по требованию

### 👥 Управление клиентами
- База клиентов с email и комментариями
- Группировка клиентов для целевых рассылок
- Индивидуальные настройки для каждого клиента

### 📊 Статистика и аналитика
- Общая статистика по рассылкам
- Отслеживание успешных/неуспешных отправок
- Визуализация эффективности кампаний
- Детальная информация по каждой попытке отправки

### 🔐 Безопасность и доступ
- Система ролей (Пользователь/Менеджер/Администратор)
- Подтверждение email при регистрации
- Восстановление пароля
- Ограничение доступа к данным

## 🛠 Технологии

- **Backend:** Django 5.2.7, Python 3.13
- **Database:** SQLite (для разработки), поддержка PostgreSQL
- **Frontend:** HTML, CSS, Bootstrap 5
- **Email:** Django SMTP, консольный бэкенд для разработки
- **Кеширование:** Django LocMemCache
- **Аутентификация:** Django Auth с кастомными профилями
